# 製品ドキュメント

## ホワイトボードシステム(WBS)

ビデオ通話とホワイトボードが使用できるネットコラボレーションツールです。

### 主な特徴

- 複数ユーザでのビデオ通話
- リアルタイムで更新される人数分のホワイトボード

### 操作マニュアル

- 簡易操作マニュアル(旧) https://github.com/czbone/product-doc/tree/master/wbs

### デモ
